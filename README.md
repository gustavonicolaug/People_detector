Here`s an Arduino project to detect people who pass or stay in front of an ultrasonic sensor

## Arduino configuration

	Ultrasonic sensor module:

	pin 5: 	ECHO
	pin 4: 	TRIGGER
	5v:		VCC
	GND:	GND

	Micro SD module:

	pin 10:	CS
	pin 11:	MOSI
	pin 12:	MISO
	pin 13:	SCK
	3.3V:	VCC
	GND:	GND

	GSM/GPRS SIM900 module:

	pin 7:	TXD
	pin 8:	RXD
	GND:	GND
